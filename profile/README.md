# 🏫 Lopatky
Tento prostor je určen výhradně pro členy týmu, kteří společně pracují na školním týmovém projektu.

## 📌 Cíl organizace
Tato organizace slouží k verzování, správě a koordinaci vývoje našeho projektu. Najdeš zde:
- Repozitáře s kódem pro backend, frontend a další části projektu
- Issues & Projects pro sledování úkolů a pokroků
- Wiki & Dokumentaci k projektu

## 🔧 Technologie, které používáme
- **Backend:** Python (Flask)
- **Frontend:** TypeScript, Nuxt.js (Vue.js), SCSS, Pinia
- **Databáze:** MySQL
- **Design:** Figma
- **Další:** Git, MySQL Workbench, Bruno/HTTPie

## 📜 Pravidla spolupráce
- Každý člen musí plnit zadané úkoly ve stanoveném termínu
- Dodržování dohodnutnuté struktury kódu a coding style
- Každý commit by měl být jasný a popsaný srozumitelnou zprávou
- Aktivní sledování Issues a GitHub Projects, pro přehled v úkolech
- Komunikace probíhá v dohodnutém kanálu (Discord, Teams)
